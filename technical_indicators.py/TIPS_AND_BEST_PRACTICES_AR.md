```
❌ تراكم التنبيهات دون تنظيم
❌ عدم تحديد الإجراء المطلوب

```

### 2️⃣ إدارة المخاطر

#### 🎓 فهم المقاييس:

**Sharpe Ratio:**
```
قيمة منخفضة (< 0.5):  ❌ استراتيجية ضعيفة
قيمة متوسطة (0.5-1.0): ⚠️ استراتيجية عادية
قيمة عالية (> 1.0):    ✅ استراتيجية جيدة
قيمة عالية جداً (> 2.0): 🌟 استراتيجية ممتازة
```

**Maximum Drawdown:**
```
أقل من -5%:  ✅ منخفض جداً - آمن
-5% إلى -15%: ⚠️ متوسط - يقبل
-15% إلى -30%: 🟠 عالي - مخاطرة
أكثر من -30%: ❌ عالي جداً - خطير
```

#### ✅ الحسابات الصحيحة:
```python
# ✅ صحيح: حساب Sharpe مع معدل خالي من المخاطر
sharpe = RiskManager.calculate_sharpe_ratio(returns, 0.05)

# ✅ صحيح: تحليل العائدات السلبية
sortino = RiskManager.calculate_sortino_ratio(returns, 0.0)

# ✅ صحيح: حساب الخسارة الأسوأ
max_dd = RiskManager.calculate_max_drawdown(returns)
```

#### 💼 قرارات بناءً على المقاييس:
```
إذا كان Sharpe > 1.0 و Max DD > -10%:
    ✅ تطبيق الاستراتيجية
    ✅ زيادة رأس المال
    ✅ توسيع عدد الأسهم

إذا كان Sharpe < 0.5 أو Max DD < -20%:
    ❌ إيقاف الاستراتيجية
    ❌ إعادة تقييم النظام
    ❌ تقليل الحجم
```

### 4️⃣ قوالب الاستراتيجيات

#### 🚀 متى تستخدم كل استراتيجية:

**1. الدعم والمقاومة:**
```
مناسبة إذا:
✅ السهم متذبذب بين مستويات محددة
✅ لديك صبر للانتظار عند المستويات
✅ تريد نسبة مخاطرة منخفضة (1:2)

غير مناسبة إذا:
❌ السهم في اتجاه قوي (لا ارتدادات)
❌ تريد ربح سريع
❌ المستويات غير واضحة
```

**2. استراتيجية الزخم:**
```
مناسبة إذا:
✅ السهم في اتجاه قوي
✅ الحجم مرتفع جداً
✅ تريد أرباح كبيرة (1:3)

غير مناسبة إذا:
❌ السوق جانبي (sideways)
❌ تريد صفقات آمنة
❌ قدرتك على المراقبة منخفضة
```

**3. المتوسط المتحرك:**
```
مناسبة إذا:
✅ تريد استراتيجية بسيطة
✅ السهم له اتجاه واضح
✅ تريد صفقات طويلة الأجل

غير مناسبة إذا:
❌ تتداول على فترات قصيرة جداً (دقائق)
❌ السوق متقلب جداً
❌ تريد ربح سريع
```

**4. التذبذب:**
```
مناسبة إذا:
✅ السهم متذبذب بشكل قوي
✅ تريد صفقات متكررة
✅ الحركة تتوقعية

غير مناسبة إذا:
❌ السهم خامل
❌ الاتجاه واضح جداً
❌ لا تستطيع المراقبة المستمرة
```

#### 💾 إنشاء استراتيجية مخصصة:

### 5️⃣ محاكاة المحفظة

#### 💼 إدارة صحيحة للمحفظة:

**حجم الموضع (Position Sizing):**
```
✅ صحيح:
   - حجم واحد = 2-5% من رأس المال
   - عدد الصفقات = 5-10 صفقات متزامنة
   - إجمالي المخاطرة = 10% من رأس المال

❌ خاطئ:
   - حجم كبير = 20% من رأس المال
   - عدد كبير جداً = 30 صفقة متزامنة
   - مخاطرة عالية = 50% من رأس المال
```

**Stop Loss و Take Profit:**
```
✅ صحيح:
   - Stop Loss = 2-5% أقل من الدخول
   - Take Profit = 4-10% أعلى من الدخول
   - النسبة = 1:2 على الأقل

❌ خاطئ:
   - بدون Stop Loss (خطر جداً!)
   - Stop Loss بعيد جداً (خسارة كبيرة)
   - Take Profit قريب جداً (لا فائدة)
   - نسبة 1:1 (لا تستحق المخاطرة)
```

**التوثيق والملاحظات:**
```python
# مثال:
trade_notes = "Breakout from 20-day resistance at 155"
# بدلاً من:
trade_notes = "nice signal"

# اكتب:
trade_notes = "Entry at 155 (20-day high), SL at 147 (2%), TP at 165 (6%)"
```

## 🔧 استكشاف الأخطاء

### مشاكل شائعة والحل:

**المشكلة: التنبيهات لا تعمل**
```
السبب المحتمل:
- البيانات غير محدثة
- الشرط غير دقيق
- التنبيه محذوف

الحل:
1. تحقق من البيانات المدخلة
2. أضف تنبيه اختباري جديد
3. راجع session_state
```

**المشكلة: قيم Sharpe غير معقولة**
```
السبب المحتمل:
- بيانات عائدات قليلة جداً
- الانحراف المعياري = صفر
- بيانات غير دقيقة

الحل:
1. استخدم عينة أكبر (100+ عائد)
2. تحقق من الحسابات
3. تأكد من صحة المدخلات
```

**المشكلة: نتائج البحث فارغة**
```
السبب المحتمل:
- المعايير صارمة جداً
- لا توجد أسهم تطابق
- البيانات قديمة

الحل:
1. رخّي المعايير تدريجياً
2. جرب معايير مختلفة
3. تحقق من البيانات
```
