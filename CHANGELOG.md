## [26.0.0] - 2026-05-16

### Added
- ✨ **نظام التنبيهات الذكية** - مراقبة فعّالة للأسعار والحجم والمؤشرات
- 📈 **إدارة المخاطر المتقدمة** - حساب VaR, Sharpe Ratio, Sortino Ratio, Maximum Drawdown
- 🔍 **مسح الأسهم الذكي** - بحث متقدم وتصفية ديناميكية للأسهم

- 📄 **مولد التقارير PDF** - توليد تقارير احترافية وسهلة التحميل
- 📖 **توثيق شاملة** - 150+ صفحة توثيق بالعربية
- 💻 **أمثلة عملية** - 30+ مثال جاهز للاستخدام
- 🎨 **واجهة احترافية** - Streamlit UI متقدمة مع Dark Mode

- تحسين تنظيم الكود

## نموذج التحديث

عند إصدار نسخة جديدة، يتم اتباع النمط التالي:

### تحديث الإصدار الكبير (Major):
- تغييرات جذرية غير متوافقة مع الإصدارات السابقة
- مثال: 25.0.0 → 26.0.0

### تحديث الإصدار الثانوي (Minor):
- إضافة ميزات جديدة متوافقة مع الإصدارات السابقة
- مثال: 26.0.0 → 26.1.0

### تحديث الإصدار التصحيحي (Patch):
- إصلاحات الأخطاء والتحسينات الصغيرة
- مثال: 26.0.0 → 26.0.1
