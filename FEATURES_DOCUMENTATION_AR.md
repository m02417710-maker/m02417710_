## 1️⃣ نظام التنبيهات الذكية (Alert Manager)

### 📋 الوصف
نظام متقدم لتعيين تنبيهات على الأسهم يراقب الأسعار والمؤشرات ويُصدر تنبيهات عند تحقق الشروط.

### 🚀 الميزات الرئيسية

#### إضافة التنبيهات
```python
alert_manager = AlertManager()
alert = alert_manager.add_alert(
    symbol="EBANK",
    alert_type="PRICE",           # PRICE, VOLUME, MACD, RSI
    condition=">",                # >, <, =
    value=150.0,                  # قيمة الشرط

)
```

#### أنواع التنبيهات
| النوع | الوصف | الاستخدام |
|-------|-------|----------|
| **PRICE** | تنبيهات السعر | عند الوصول لسعر معين |
| **VOLUME** | تنبيهات الحجم | عند تجاوز حجم معين |
| **MACD** | تنبيهات MACD | عند تقاطع المؤشر |
| **RSI** | تنبيهات RSI | عند مستويات التشبع |

#### الواجهة المستخدم
- ✅ إضافة تنبيهات جديدة
- ✅ عرض التنبيهات النشطة
- ✅ حذف التنبيهات
- ✅ تحديد حالة التنبيهات (مفعل/مُفعّل)

## 2️⃣ إدارة المخاطر المتقدمة (Risk Manager)

### 📊 المقاييس المحسوبة

#### 1. Value at Risk (VaR)
```python
var = RiskManager.calculate_var(returns, confidence=0.95)
```
- **التعريف**: أقصى خسارة متوقعة بمستوى ثقة 95%
- **الاستخدام**: تحديد الحد الأقصى للخسارة المحتملة
- **الصيغة**: يتم حسابها من التوزيع التاريخي للعائدات

#### 2. Sharpe Ratio
```python
sharpe = RiskManager.calculate_sharpe_ratio(returns)
```
- **التعريف**: العائد الزائد لكل وحدة مخاطرة
- **الاستخدام**: مقارنة الاستثمارات المختلفة
- **الصيغة**: (متوسط العائد - معدل خالي من المخاطر) / الانحراف المعياري

#### 3. Sortino Ratio
```python
sortino = RiskManager.calculate_sortino_ratio(returns)
```
- **التعريف**: مشابه Sharpe لكن يركز على المخاطر السلبية فقط
- **الاستخدام**: تقييم الاستراتيجيات ذات التوزيعات غير المتماثلة
- **الفائدة**: أفضل من Sharpe للعائدات ذات الذيل السميك

#### 4. Maximum Drawdown
```python
max_dd = RiskManager.calculate_max_drawdown(returns)
```
- **التعريف**: أكبر انخفاض من قمة إلى قاع
- **الاستخدام**: قياس أقسى فترة خسارة
- **الأهمية**: حرج جداً لنفسية المتداول

#### 5. تصنيف مستوى المخاطر
```python
risk_level, risk_color = RiskManager.get_risk_level(volatility, drawdown, var)
```

**التصنيف:**
- 🔴 **عالي جداً**: Volatility > 30% أو DD < -20% أو VaR < -15%
- 🟠 **عالي**: Volatility > 20% أو DD < -10% أو VaR < -8%
- 🟡 **متوسط**: Volatility > 10% أو DD < -5% أو VaR < -3%
- 🟢 **منخفض**: باقي الحالات

### 📈 الرسوم البيانية
- رسم العائدات التراكمية
- عرض توزيع العائدات
- تصور Drawdown

## 3️⃣ مسح الأسهم الذكي (Stock Screener)

### 🎯 معايير البحث المتاحة

#### البحث حسب السعر
```python
criteria = {
    'price_min': 50.0,
    'price_max': 200.0
}
results = screener.filter_by_criteria(criteria)
```

#### البحث حسب التغير اليومي
```python
criteria = {
    'change_min': 1.0,      # تغير موجب أكثر من 1%
    'change_max': 5.0       # تغير حتى 5%
}
```

#### البحث حسب الحجم
```python
criteria = {
    'volume_min': 2000000   # حجم أكثر من مليونين
}
```

### 🚀 البحث عن الاختراقات
```python
breakouts = screener.find_breakouts(threshold=0.05)
```
- يجد الأسهم بتحركات كبيرة
- مرتب حسب حجم التحرك

### 🎯 البحث عن الدعم والمقاومة
```python
support_resistance = screener.find_support_resistance()
```
**الحقول المرجعة:**
- `support`: مستوى الدعم المحسوب
- `resistance`: مستوى المقاومة
- `distance_to_support`: المسافة بالنسبة المئوية
- `distance_to_resistance`: المسافة بالنسبة المئوية

### 💡 حالات الاستخدام
1. **البحث عن أسهم بأسعار معينة**: للمستثمرين الباحثين عن فئة سعرية
2. **البحث عن الأسهم النشطة**: للمتداولين الذين يريدون حركة
3. **البحث عن الأسهم بحجم عالي**: مؤشر على النشاط والسيولة

## 4️⃣ قوالب الاستراتيجيات (Strategy Manager)

### 📚 الاستراتيجيات المدمجة

#### 1️⃣ استراتيجية الدعم والمقاومة
```
الوصف: تداول من مستويات الدعم والمقاومة
قاعدة الدخول: ارتداد من الدعم أو اختراق المقاومة
قاعدة الخروج: الوصول للهدف أو كسر المستوى
نسبة المخاطرة والعائد: 1:2
المؤشرات: Support/Resistance, Volume, MA
```

#### 2️⃣ استراتيجية الزخم
```
الوصف: تتابع اتجاهات قوية
قاعدة الدخول: كسر مستوى عالي مع حجم مرتفع
قاعدة الخروج: انعكاس الاتجاه أو الوصول للهدف
نسبة المخاطرة والعائد: 1:3
المؤشرات: RSI, MACD, Volume
```

#### 3️⃣ استراتيجية المتوسط المتحرك
```
الوصف: تتابع الاتجاه باستخدام المتوسطات
قاعدة الدخول: عندما يقطع السعر المتوسط من الأسفل
قاعدة الخروج: عندما يقطع المتوسط من الأعلى
نسبة المخاطرة والعائد: 1:2.5
المؤشرات: MA, Close Price
```

#### 4️⃣ استراتيجية التذبذب
```
الوصف: الشراء والبيع بين المستويات
قاعدة الدخول: السعر يقترب من الحد الأدنى
قاعدة الخروج: السعر يقترب من الحد الأعلى
نسبة المخاطرة والعائد: 1:1.5
المؤشرات: RSI, Bollinger Bands, Support/Resistance
```
