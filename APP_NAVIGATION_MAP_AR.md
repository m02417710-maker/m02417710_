### قسم إدارة المخاطر (Risk Management)
```
⚖️ إدارة المخاطر المتقدمة
├─ 📊 مقاييس المخاطر الرئيسية (4 بطاقات)
│  ├─ 1️⃣ Value at Risk (95%)
│  │  └─ يعرض: var_95 = -0.XX%
│  │
│  ├─ 2️⃣ Sharpe Ratio
│  │  └─ يعرض: sharpe = X.XX
│  │
│  ├─ 3️⃣ Sortino Ratio
│  │  └─ يعرض: sortino = X.XX
│  │
│  └─ 4️⃣ Maximum Drawdown
│     └─ يعرض: max_dd = -X.XX%
│
├─ 📈 مستوى المخاطر الإجمالي
│  └─ عرض: 🟢/🟡/🟠/🔴 + الوصف
│
└─ 📉 رسم بياني
   ├─ العائدات التراكمية (Cumulative Returns)
   ├─ المحور الأفقي: الفترة (Period)
   ├─ المحور العمودي: العائد (Return)
   └─ تفاعلي (Hover للقيم)

البيانات المستخدمة: محاكاة عائدات يومية

الحسابات:
- VaR: من التوزيع التاريخي للعائدات
- Sharpe: (E(R) - Rf) / σ
- Sortino: نسخة Sharpe للعائدات السلبية
- Max DD: أكبر انخفاض من القمة للقاع
```

**الاستخدام**:
1. اعرض جميع مقاييس المخاطر
2. قارن الاستراتيجيات المختلفة
3. ابحث عن نقاط الضعف
4. استخدم للقرارات الاستثمارية

### قسم مسح الأسهم (Stock Screener)
```
🔍 مسح الأسهم الذكي
├─ 🎯 معايير البحث (4 أعمدة)
│  ├─ السعر الأدنى (price_min)
│  ├─ السعر الأقصى (price_max)
│  ├─ التغير الأدنى % (change_min)
│  └─ الحجم الأدنى (volume_min)
│
├─ ✅ النتائج (جدول ديناميكي)
│  └─ يعرض:
│     ├─ السهم (Symbol)
│     ├─ السعر (Price)
│     ├─ التغير % (Change %)
│     └─ الحجم (Volume)
│
├─ 🚀 الاختراقات
│  └─ قائمة الأسهم بتحركات > 2%
│     ├─ مرتب حسب حجم التحرك
│     ├─ إشارات: 📈 للموجب
│     └─ إشارات: 📉 للسالب
│
└─ 💡 المسح الضوئي المتقدم
   ├─ البحث عن الدعم والمقاومة
   ├─ تحديد الأسهم القريبة من المستويات
   └─ تقارير الفرص

البيانات: قاموس أسهم مع الأسعار والحجم
```

**الاستخدام**:
1. اختر المعايير المطلوبة
2. شاهد النتائج مباشرة
3. ركز على الاختراقات
4. ابحث عن الفرص
